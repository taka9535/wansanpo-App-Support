# わん散歩 - サポートページ

愛犬との散歩を楽しく記録・管理できるアプリ「わん散歩」のサポートページです。

## 主な機能

- GPSで正確な散歩ルートを記録
- 散歩ごとのメモ機能
- 週間・月間・年間の散歩記録を集計
- 愛犬のプロフィール管理
- 散歩中のイベント（おしっこ・うんち・挨拶）を地図上に記録
- 天気・気温・タグで詳細なメモを記録
